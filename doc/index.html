libebb

libebb is a lightweight HTTP server library for C. It abstracts the socket handling and request parsing. It handles persistant connections, pipelined requests, chunked uploads, or any other HTTP/1.1 complient junk you might throw at it.

  • there are no allocations made
  • the libev event loop is used.
  • it is single threaded
  • the parser is nearly optimally fast. it is a state machine written in Ragel

libebb does not support SSL, but may in the future.

libebb is intended to be used as the central engine in all types of web servers: load balancers (like perlbal), embeded servers (like Mongrel), or even full service servers like Nginx.

Usage

Functionality is provided by implmenting many callbacks.
